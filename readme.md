# EarthLiveSharp
项目来自[Earth Live Sharp ](https://github.com/bitdust/EarthLiveSharp)
图片均来自向[日葵-8號即時網頁](http://himawari8.nict.go.jp/)

这是我用python写的脚本文件,ubuntu下测试正常,直接运行 python wallpaper.py

每十分钟更新一次照片,并自动设置为壁纸.可加入开机启动项.
